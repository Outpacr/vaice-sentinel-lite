# Contributing
- Submit all prompts as `*.yaml` files, compliant with `schemas/prompt.schema.json`.
- Add the Apache-2.0 header to the top of each file.
- No binaries, no model weights, no secrets.
- PRs must pass the linter and schema checks; **no** runtime tests are executed.

**Commit Convention (short):**
- `feat(prompt): ...` for a new prompt
- `fix(prompt): ...` for a correction
- `docs: ...` for documentation changes
